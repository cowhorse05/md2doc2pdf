# 不同光照条件对硅太阳能电池输出特性的影响

## 实验目的

1. 研究光照强度对太阳能电池开路电压和短路电流的影响
2. 测量不同负载下的 I-V 特性曲线
3. 计算电池的填充因子（Fill Factor）和转换效率

## 实验原理

太阳能电池的光生电流与光照强度成正比：

$$I_{ph} = \frac{q \eta P_{in}}{h \nu}$$

其中 $q$ 为电子电荷，$\eta$ 为量子效率，$P_{in}$ 为入射光功率。

I-V 特性方程：

$$I = I_{ph} - I_0 \left[\exp\left(\frac{q(V + IR_s)}{nkT}\right) - 1\right] - \frac{V + IR_s}{R_{sh}}$$

## 实验步骤

1. 搭建电路：太阳能电池 → 电流表（串联）→ 可变电阻 → 电压表（并联）
2. 设置光照强度为 200/400/600/800/1000 W/m² 五组
3. 每组测量 15 个负载点的 I-V 数据
4. 记录环境温度（保持在 25±2°C）
